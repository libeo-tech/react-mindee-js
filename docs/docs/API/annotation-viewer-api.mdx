# AnnotationViewer

:::info
**AnnotationViewer** use **[KonvaJs](https://konvajs.org)** (HTML5 2d canvas js library for desktop and mobile applications)
:::

### data

- Type : `object`
- Required : `true`
- Interface

```tsx
type AnnotationData = {
  image: string
  orientation: 0 | 90 | 180 | 270
  shapes: AnnotationShape[]
}

type AnnotationShape = {
  id: string
  coordinates: number[][]
  config?: LineConfig
}
```

> **image** property should be either an URL object or a base64. If you have an object of type [File](https://developer.mozilla.org/fr/docs/Web/API/File) ( typically generated by file upload libraries) instead you may need to use [URL.createObjectURL()](https://developer.mozilla.org/fr/docs/Web/API/URL/createObjectURL) to generate a valid URL object.

:::caution

you must provide a unique **id** to each shape for optimization matters.

:::

Check [LineConfig](https://konvajs.org/api/Konva.Line.html#main) for more params.

---

### options

- Type : `Object`
- Interface :

```ts
interface AnnotationViewerOptions {
  shapeConfig?: LineConfig
  selectionRectConfig?: RectConfig
  enableSelection?: boolean
  onMouseEnter?: (polygon: Line) => void
  onMouseLeave?: (polygon: Line) => void
  onClick?: (polygon: Line) => void
  zoom?: {
    modifier: number
    max: number
    defaultZoom: number
  }
}
```

> - Object of options to change default style and behavior. Here you can pass a custom style to all shape as you may do for each one.

#### Check [Customization](/docs/customization) section for more details.
